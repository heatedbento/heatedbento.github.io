a introduction to TwitchOnline

TwitchOnline is a website that allows you to check if your favourite streamer is live or not in adition to that you can check if a disired username is taken and given information on it. I strive for minimalism funtionality and user freindliness. So try signing up so that you can connect your account to multiple devices and see when your favourite streamers are online.

ctrl/cmd+c - check if streamer is live
ctrl/cmd+u - get information on username
